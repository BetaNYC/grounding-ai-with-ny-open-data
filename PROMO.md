We're publishing a new open resource for anyone building AI tools on New York's civic data: **grounding-ai-with-ny-open-data**. AI agents are only as trustworthy as what they're grounded in, and government open data is one of the clearest ways to keep an agent's answers checkable against the real public record instead of a guess. The repo includes ready-to-use configs for connecting AI tools to both the NYC and NYS open data portals, guides on which government data source actually has what you're looking for, and a directory of open data portals run by counties and municipalities across the state. It's an early, working draft, and we'll keep building it out. [Check it out on GitHub →](https://github.com/BetaNYC/grounding-ai-with-ny-open-data)
